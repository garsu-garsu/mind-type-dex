# 마음 유형 도감 (mind-type-dex) — 기획 1-pager

> 매일 한 번, 4문항 짧은 마음 테스트로 **오늘의 나는 어떤 유형인지** 알아보고,
> 16종 '마음 유형' 카드를 도감에 모으는 **비게임 심리 테스트 앱**.

- **slug**: `mind-type-dex` · **displayName**: 마음 유형 도감 (8자)
- **카테고리**: 비게임(진단/테스트 콘텐츠) · **타깃 톤**: 발랄·귀여운 감성(실유저 19+)
- **로그인**: 없음(익명/로컬, localStorage). 친구 비교는 **공유 카드**로 → 추가 개인정보 0 → 연결끊기 엔드포인트 불필요
- **백엔드**: 없음(클라이언트 전용). KST 일일 리셋만 `getServerTime` 사용

## 1. 비게임 코어 (게임 아님)

**코어 동사 = 진단/생성.** 사용자는 "이기거나" "점수 경쟁"하지 않아요. 매일 짧은 심리/취향 테스트에 답하면 **결과 유형 카드(리포트)** 를 생성·수집할 뿐.

- 4개 **이분 축**(각 1문항, 총 4문항):
  1. 에너지 — `바깥형` ↔ `안쪽형`
  2. 결정 — `마음형(감정)` ↔ `머리형(논리)`
  3. 하루 리듬 — `즉흥형` ↔ `계획형`
  4. 시선 — `지금형` ↔ `멀리형`
- 4비트 조합 → **2⁴ = 16종 유형**으로 **결정적 매핑**(랜덤·확률 뽑기 아님 → 사행성 표면 0).
- 각 유형 = 동물 캐릭터(이모지) + 한 줄 요약 + **상세 해석**(광고 게이트로 해금).
- 매일 문항 문구는 회전(콘텐츠 회전)되지만 축은 동일 → "오늘의 나"는 그날 답에 따라 달라짐 → 도감을 채울 동기.

> §1.0 비게임 판별 통과: ① 잘해서 이기는 코어 아님 ② 반사신경/타이밍 아님 ③ 육성/클리어 아님.

## 2. 코어 루프

```
[오늘의 테스트 4문항]
        ↓
[결과: 오늘의 유형 카드 생성]  ← 전면(I) 자연 경계 1회
        ↓
[유형 도감에 슬롯 채움 (n/16)]
        ↓
┌────────────────────────────────────────────┐
│  상세 해석 보기 → 📺 광고 보고 상세 해석 해금 (R)   │ ← 고욕구 지점, 결정적
│  오늘 한 번 더    → 📺 광고 보고 오늘 테스트 하나 더 (R)│
└────────────────────────────────────────────┘
        ↓
[연속 기록 +1 / 분석가 등급 갱신]
        ↓
[KST 자정 리셋 + (동의 시) 일일 리마인드 푸시] → 다음날 복귀
```

## 3. 광고 배치 지도 (전부 결정적 보상 · 화면당 배너 1개)

| 액션 | 포맷 | 게이트 카피(결정적) |
|---|---|---|
| 결과 화면에서 상세 해석 보기 | 보상형 `R` | `📺 광고 보고 상세 해석 해금` |
| 오늘 테스트 1회 추가 | 보상형 `R` | `📺 광고 보고 오늘 테스트 하나 더` |
| 연속 기록 끊겼을 때 복구 | 보상형 `R` | `📺 광고 보고 연속 기록 지키기` |
| 결과 카드 공개 직후(경계) | 전면 `I` | 세션당 ≤1 (빈도 3회당 1) |
| 홈/도감 하단 | 배너 `B` | 화면당 1개 (`variant:'card'`) |

❌ 금지 준수: 확률↑·리롤·더 좋은 상자·황금상자 없음 / 배너 스택 없음 / 전면 실행·종료·도중 호출 없음.

## 4. 가상 보상 경제 (현금 0)

- **유형 도감**: 16종 수집 슬롯(순서대로 채우는 비랜덤 진행 → 사행성 면역).
- **연속 기록(streak)**: 연속으로 테스트한 일수. 끊기면 `광고 보고 연속 지키기`로 1회 복구.
- **분석가 등급**: 수집한 유형 수 기준 등급(새싹 분석가 → … → 마음 박사). 칭호 표시.
- **코스메틱(후순위)**: 카드 테마 색. → 매출 안정 후 토스포인트 전환 지점(지금은 가상만).

## 5. 리텐션 · 바이럴

- **KST 일일 리셋**: `getServerTime`(서버 Unix ms)+9h → 날짜 키. 미지원 시 기기 시각 폴백.
- **연속 기록**: 손실회피 → 강한 복귀.
- **동의형 일일 푸시**: `requestNotificationAgreement(templateCode)`(템플릿 코드 있을 때만), `notify_consent` 로깅.
- **공유 카드**: 오늘의 유형 결과를 `getTossShareLink`+`share`로 공유("내 마음 유형은 ○○ 🦉").

## 7. 심사 차별점 · 안전

- 토스 네이티브(만보기·행운퀴즈·**출석 앱테크**·소비분석)와 **메커니즘 차별**: 코어가 "심리 유형 진단/수집"이라 겹치지 않음. "출석/연속출석" 표현 금지 → **"연속 기록"**.
- 사행성 금칙어(베팅·당첨·추첨·잭팟·도박·뽑기·올인·리롤) 0건(카피·인앱·스토어).
- 만남·연애·1:1채팅 프레이밍 0. 의료·단정 표현 금지(웰빙/재미 콘텐츠).
- 위치 미사용(콘텐츠 데이터). 로그인/추가 개인정보 0 → 연결끊기 엔드포인트 불필요.
